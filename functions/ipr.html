Ссылка на первоисточник: Формирование ИПР в ЕИС

Примечание: Для каждого сотрудника разрабатывается "Индивидуальный план развития". *По сотрудникам на испытательном сроке не готовим.

Срок проведения: Раз в квартал (в конце квартала).

Описание

Работа с индивидуальными планами развития является эффективным инструментом для развития сотрудников и обеспечения высокой производительности в организации. Она позволяет создать условия для постоянного роста и достижения максимального потенциала каждого сотрудника.

Работа в ЕИС

1. Откройте список документов "Индивидуальные планы развития"

Путь: Зарплата и управление персоналом > Индивидуальные планы развития

2. Установите отбор по сотруднику

3. Подведите итоги работы по Индивидуальному плану развития

Откройте последний по дате документ, работа с которым не завершена (Статус – В работе).

3.1 Установите статус по каждому мероприятию:

  • Завершено – мероприятие выполнено.
  • Не завершено – на момент подведения итогов работы по индивидуальному плану развития.
  • Отменено – принято решение о том, что не следует выполнять развивающее мероприятие.

3.2 Вносите в поле Комментарий что сделано в рамках развивающего мероприятия.

3.3 Установите Статус документа: Завершен

4. Разработайте новый индивидуальный план развития на следующий квартал

4.1 Создайте новый документ "Индивидуальный план развития". В верхней части документа укажите:

  • период, на который создается план
  • специализацию
  • текущий уровень квалификации
  • планируемый уровень
  • Установите статус документа Черновик

4.2 На странице "Плановые мероприятия" подготовьте список развивающих мероприятий.

Заполнить табличную часть можно вручную по кнопке Добавить. Также можно заполнить табличную часть компетенциям сотрудника с помощью подменю "Заполнить компетенции".

Подготовительная работа:
  • Анализ текущей ситуации: на основе результатов оценки компетенций и подготовленных матриц компетенций оцените сильные и слабые стороны каждого сотрудника. Проанализируйте потребности в обучении и развитии. Изучите текущие задачи и проекты.
  • Определение целей и задач: Совместно с каждым сотрудником определите цели развития. Разбейте цели на конкретные задачи. Установите сроки выполнения.
  • Разработка плана развития: Составьте план обучения и развития. Включите тренинги, курсы, наставничество, чтение специальной литературы, участие в проектах. Определите ресурсы и бюджет.
При заполнении ИПР необходимо:
  1. Проверить наличие необходимого материала (ссылки на диск, индиго или веб-ресурс) или разработать собственное мероприятие.
  2. Включить в ИПР обязательное мероприятие — выработку.
  3. Включить в ИПР мероприятие по развитию личностных компетенций.

В каждом ИПР должно быть предусмотрено одно мероприятие по развитию личностных компетенций (справочник «Компетенции» — СИСТЕМА КОМПЕТЕНЦИЙ 1С-РАРУС — ЛИЧНОСТНЫЕ).

На индиго есть хорошие курсы по данной теме: indigo.rarus.ru

При заполнении развивающих мероприятий укажите:

Мероприятие, направленное на развитие компетенций. Выбирайте развивающие мероприятия из справочника "Развивающие мероприятия".

При необходимости добавляйте новые развивающие мероприятия в папку своего офиса.

В поле Комментарий можно указать дополнительные сведения о мероприятии: ссылки, описание.

Статус мероприятия. Используйте только перечисленные ниже статусы:

  • Новое – мероприятие включено в план, но не запланирована работа по нему.
  • Запланировано – определены сроки проведения мероприятие.
  • В работе – работа по мероприятию начата, имеет смысл отмечать только для длительных по времени мероприятий.
  • Завершено – мероприятие выполнено.
  • Не завершено – на момент подведения итогов работы по индивидуальному плану развития.
  • Отменено – принято решение о том, что не следует выполнять развивающее мероприятие.
  • Перенесено – не используется.

Дата начала, Дата завершения, Продолжительность, Объем часов (трудоемкость) – плановые значения по развивающему мероприятию.

  • Стоимость следует указывать только при использовании внешних ресурсов для обучения, за которые компания платит.
  • Вид сертификата. Указывайте, если развивающее мероприятие нацелено на получение сертификата.

Компетенция, Плановая оценка. Указывайте, если развивающее мероприятие нацелено на получение конкретной компетенции.

5. Заполните пожелания сотрудника и руководителя по развитию компетенций

На странице "Дополнительно".

6. По окончании подготовки ИПР установите статус "На согласовании" и сохраните документ.

7. Обсудите и согласуйте ИПР с сотрудником

После того, как руководитель и сотрудник договорились о работе по индивидуальному плану развития, установите статус документа: В работе.

8. Работа с ИПР в течение квартала

Мониторинг и оценка:
  • Регулярно проверяйте прогресс выполнения задач.
  • Проводите промежуточные оценки.
  • Корректируйте план при необходимости.
Поддержка и мотивация:
  • Обеспечьте доступ к необходимым ресурсам.
  • Поддерживайте мотивацию сотрудников.
  • Поощряйте достижения и успехи.
Обратная связь и корректировка:
  • Организуйте регулярные встречи для обсуждения прогресса.
  • Собирайте обратную связь от сотрудников.
  • Вносите необходимые изменения в план.
