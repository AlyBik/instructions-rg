Общая информация

Примечания: АРМ планирования в ЕИС, раз в неделю контролировать, два раза в месяц актуализировать. У сотрудника не обязательно должна быть 100% загрузка в АРМ планирования.

Срок проведения: В последний день месяца – на полгода вперед. До 13 числа – на полгода вперед. Актуализация к среде каждой недели, так как в среду ОПЗС с руководителем отдела ОМКС анализируют.

Описание

Планирование загрузки сотрудников позволяет:

  • Распределить нагрузку между сотрудниками таким образом, чтобы избежать перегрузки одних сотрудников и недостаточной загрузки других.
  • Определить загруженность каждого сотрудника.
  • Планировать загрузку сотрудников на основе их навыков, компетенций и текущей загруженности.

Работа в ЕИС

1. Откройте АРМ планирования

Путь: Главное > Рабочее время (план) > АРМ планирования.

2. Установите группировку

Можно установить дополнительную группировку по сервису и клиентам, а также другие группировки для анализа загрузки.

3. Установите необходимые отборы

4. Установите период планирования загрузки

Совет: При установке периода планирования удобно выводить предыдущий месяц, чтобы заполнить план, копируя строки. Это помогает, например, если в справочнике "Клиенты" есть позиции с одинаковыми наименованиями, и сложно выделить нужный элемент справочника.

5. Проанализируйте планируемую и фактическую загрузку

Для этого можно воспользоваться отчетом "Отчет план-фактного анализа трудозатрат" (или ОСВ).

Предлагаемый вариант отчета: Отчет план-фактного анализа трудозатрат
Ссылка: e1cib/data/Справочник.ВариантыОтчетов?ref=a1094cd98f59d07b11efcb7590693cf5

Важно: Сравнивая Часы по плану и Часы по работам по отчету, можно оценить результаты работы сотрудника и спрогнозировать нагрузку сотрудника в следующем периоде.

6. Заполните загрузку по каждому сотруднику

6.1 Метод копирования строк предыдущего месяца

Выделите необходимую строку и выберите команду "Скопировать".

В открывшемся окне скорректируйте Период и Загрузку. Загрузку можно устанавливать двумя способами: корректируя процент загрузки или количество часов.

Далее необходимо нажать "Записать и закрыть"

6.2 Метод добавления новой строки

Выберите сотрудника и период для внесения данных о загрузке.

В табличной части нажмите "Добавить". В открывшемся окне заполните:

  • Тип работ (Часто используемые: Сервис/Проект/Внутренняя работа/Исключение)
  • Вид работ (Часто используемые для типа работ Сервис: Поддержка, обработка обращений)
  • Клиент
  • Сервис

Заполните Период и Загрузку. Загрузку можно устанавливать двумя способами: корректируя процент загрузки или количество часов.

В этом окне отображается планируемая загрузка на установленный период, а также внесенные ранее данные о загрузке и фактическое отклонение (недогруз или превышение) в зависимости от установленного максимума на этот период.

7. Примеры заполненных загрузок по сотрудникам

Пример 1: У сотрудника планируется 100 % загрузка

Пример 2: Планируемая нагрузка для сотрудника с внутренними задачами

Для сотрудника, за которым закреплены внутренние задачи (например, для руководителей — административная работа, для сотрудников на испытательном сроке — обучение).
