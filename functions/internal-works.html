Описание

Контроль содержания внутренних работ сотрудников позволяет:

  • Убедиться, что все внутренние работы выполняются в соответствии с установленными процедурами, правилами и нормами. Это гарантирует качество и эффективность работ.
  • Оперативно обнаруживать возникающие проблемы и недостатки в процессе выполнения работ.

Работа в ЕИС

Предлагаемый вариант анализа внутренних работ при помощи отчета: Отчет план-фактного анализа трудозатрат

Ссылка: e1cib/data/Справочник.ВариантыОтчетов?ref=a1094cd98f59d07b11efcb7590693cf5

1. В сформированном отчете выбрать внутренние работы по колонке Часы по работам

2. Расшифровать строку отчета по Комментарию

3. Выполнить анализ внутренних работ по сотруднику

При необходимости обсудить с сотрудником о причинах превышения количества часов по внутренним работам и варианты уменьшения часов по внутренним работам в дальнейшем.

Важно: Максимальное количество допустимых часов по внутренним работам по сотрудникам ОМКС: 16
